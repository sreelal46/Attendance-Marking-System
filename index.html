üìç Attendance Marking Rules

  • Who is marked? Only students whose names contain the Batch Name set in Settings (e.g., "BCR78").
  • How are names matched? We handle spelling variations (e.g., "Mohammed" vs "Muhammed") and ignore spaces, so "MohammedShanif" matches "Mohammed Shanif".
  • Minimum Time: A student must be in the call for at least 48 minutes to be marked present.
  • What shows in the report? Clean names only. Batch codes and technical suffixes are automatically removed.
